## Tauri Commands (40+)

### App
- `init_app` — инициализация, определение HERMES_HOME
- `detect_instances` — поиск Python инсталляций
- `check_python_path` — проверка пути к Python
- `get_app_version` — версия приложения

### Connection
- `get_connection_config` — получить конфиг подключения
- `set_connection_config` — сохранить конфиг
- `test_connection` — проверить подключение (local/remote/SSH)

### Gateway
- `start_gateway_cmd` — запустить hermes gateway
- `stop_gateway_cmd` — остановить gateway
- `gateway_status_cmd` — статус gateway
- `get_gateway_port_cmd` — порт gateway

### Chat
- `send_message_cmd` — отправить сообщение (локально/удалённо/через SSH)

### Sessions
- `list_sessions_cmd` — список сессий из SQLite
- `get_session_messages_cmd` — история сообщений
- `search_sessions_cmd` — поиск сессий
- `delete_session_cmd` — удалить сессию
- `get_session_stats_cmd` — статистика

### Profiles
- `list_profiles_cmd` — список профилей
- `create_profile_cmd` — создать профиль
- `delete_profile_cmd` — удалить профиль
- `set_active_profile_cmd` — установить активный профиль

### Models
- `list_models_cmd` список моделей
- `add_model_cmd` добавить модель
- `remove_model_cmd` удалить модель
- `update_model_cmd` обновить модель

### Skills
- `list_installed_skills_cmd` — список установленных скиллов
- `get_skill_content_cmd` — содержимое скилла
- `install_skill_cmd` — установить скилл
- `uninstall_skill_cmd` — удалить скилл

### Config
- `get_env_cmd` — переменные окружения
- `set_env_cmd` — установить переменную
- `get_model_config_cmd` — конфигурация модели
- `set_model_config_cmd` — установить конфигурацию

### SSH
- `start_ssh_tunnel_cmd` — запустить SSH туннель
- `stop_ssh_tunnel_cmd` — остановить туннель
- `ssh_tunnel_status_cmd` — статус туннеля

## Chat Events

События от Rust backend через `listen("chat_event")`:

```typescript
type ChatEvent =
  | { type: "token"; content: string }
  | { type: "reasoning"; content: string }
  | { type: "tool_start"; name: string; tool_call_id: string }
  | { type: "tool_complete"; name: string; tool_call_id: string; output: string; duration_ms: number }
  | { type: "approval_request"; request_id: string; tool_name: string; tool_input: string; action: string; command_class: string }
  | { type: "pipeline_status"; backend: string; model?: string; tokens_used?: number; tokens_limit?: number; cost_usd?: number }
  | { type: "done"; session_id?: string }
  | { type: "error"; message: string }
  | { type: "status"; status: string }
```

## Connection Modes

### Local Mode
- Запуск hermes gateway как child process
- Per-profile port allocation (8642 + hash)
- Health polling каждые 15 сек

### Remote Mode
- HTTP/HTTPS к удалённому hermes API
- URL + API key из connection config
- Использует `/v1/chat/completions`

### SSH Mode
- SSH tunnel: `ssh -N -L {local}:127.0.0.1:{remote} user@host`
- Remote execution для file operations
- Все операции проксируются через SSH

## Известные ограничения

- tui_gateway protocol: пока используется API mode (`/v1/chat/completions`), не raw JSON-RPC stdin/stdout
- Windows сборка: требуется Windows хост, нет cross-compile из Linux
- AppImage GLIBC: собирать на старейшей target OS
- MCP catalog: возвращает пустой список (заглушка)
- Cron jobs: хранение в JSON, выполнение через gateway не реализовано
